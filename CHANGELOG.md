# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-19

### Added
- Initial release
- **Classic Training Zones Collection** (16 workouts)
  - Recovery (2 workouts)
  - Endurance (2 workouts)
  - Tempo (2 workouts)
  - Sweet Spot (4 workouts)
  - Threshold (2 workouts)
  - VO2max (4 workouts)
- **4DP-Style Collection** (14 workouts)
  - FTP/Endurance (4 workouts)
  - MAP/Attack (4 workouts)
  - AC/Breakaway (3 workouts)
  - NM/Sprint (3 workouts)
- Comprehensive documentation
  - Main README
  - Collection-specific READMEs
  - Comparison guide
  - FAQ
  - Contributing guidelines
- GitHub issue templates
- MIT License

### Notes
- All workouts are 20-35 minutes duration
- Power-based (% of FTP)
- Compatible with TrainingPeaks, Wahoo, Zwift
- Tested on Wahoo KICKR
